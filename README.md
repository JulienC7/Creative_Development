# Creative Dev - Mourning

Projet réalisé avec **Vue.js**.
L'objectif était d'expérimenter sur le **Creative Development** en racontant les étapes du deuil (Colère, Déprime, Sérénité) sans passer par du gros texte, mais juste avec l'ambiance visuelle et sonore.


## Ce que j'ai mis dedans

Le site traverse 3 phases :

1.  **La Colère :** Je voulais un truc agressif. J'ai utilisé des sons plutôt "horreur" et du rouge pour montrer la saturation mentale et la folie.
2.  **L'Abîme (Déprime) :** Le contraste total. J'ai fais un effet de "ghosting" en CSS sur un tourbillon pour donner l'impression de vide et de perte de repères.
3.  **L'Aube (Sérénité) :** Le retour au calme. Juste de la lumière et une nappe sonore pour finir l'expérience proprement.

## Tech

* **Vue 3 + Vite**
* Beaucoup de **CSS natif** (animations, keyframes) pour les effets visuels.
